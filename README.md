# Calculator in C

A simple command-line calculator built using the C programming language.  
This project performs basic mathematical operations using a menu-driven program.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Square of a number
- Square root of a number
- Exit option

## Technologies Used

- C Programming
- Standard Libraries:
  - stdio.h
  - math.h
  - stdlib.h

## How It Works

The program displays a menu of operations.  
The user selects an operation by entering a number, then inputs values to perform the calculation.
